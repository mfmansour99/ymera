# ============================================================================
# TROUBLESHOOTING
# ============================================================================
#
# If installation fails:
#
# 1. Upgrade pip: pip install --upgrade pip setuptools wheel
# 2. Install build tools (Linux): apt-get install build-essential python3-dev
# 3. Install build tools (Mac): xcode-select --install
# 4. For cryptography issues: Install libffi-dev libssl-dev
# 5. For psycopg2 issues: Install postgresql-dev or use psycopg2-binary
#
# Common issues:
#
# - "No matching distribution": Check Python version (3.10+ required)
# - "Microsoft Visual C++": Install Visual Studio Build Tools (Windows)
# - "Command errored out": Install missing system dependencies
# - Permission denied: Use virtual environment or --user flag
